# EVA Release Smoke Checklist

## Auth
- Sign up
- Verify by email code or magic link
- Sign in as returning user
- Confirm unverified users are routed back into verification

## Onboarding and workspace
- Complete onboarding without loops
- Confirm dashboard opens after onboarding
- Confirm workspace recovery appears instead of bouncing back to onboarding

## Finance loop
- Log spending in chat
- Check dashboard and spending history for the same event
- Generate a financial statement
- Upload CSV and review draft transactions
- Verify receipt forwarding guidance and address flow

## Support and stability
- Open key support article links
- Confirm no blank deploy
- Confirm offline and error states are readable
